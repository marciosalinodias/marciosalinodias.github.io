Html Attributes

Permite configurar informações adicionais para as tags:

  • Todas as tags possuem atributos
  • Adicionam informações às tags
  • São declarados na tag de abertura
  • Podem vir em dois formatos: chave="valor" ou valor
  • O valor pode estar entre aspas simples, duplas ou diretamente no código

Attributes="Global" (pt. 1)

Existem diversos atributos que são específicos de algumas tags.
Porém há muitos atributos globais, cuja função é a mesma para todas as tags.

accesskey Define qual tecla será usada ara acesso rápido do elemento.
class Define classes para o elemento (podendo relacionar com classes css). Classes podem se repetir.
contenteditable Determina se o conteúdo é editavel.
contextmenu Define um menu de contexto para o elemento (botão direito do mouse).
data-* Define valores privados para os elementos, não os afetando visualmente.
dir Determina a direção do texto.
draggable Define se o elemento pode ser arrastado (draggable).
dropzone Define a area de drop dos elementos draggable.

Attributes="Global" (pt. 2)

Existem diversos atributos que são específicos de algumas tags.
Porém há muitos atributos globais, cuja função é a mesma para todas as tags.

hidden Torna o elemento irrelevante, escondido.
id Define um identificador para o elemento (único na página).
lang Seta a linguagem do elemento. Usado por ferramentas de acessibilidade.
spellcheck Ativa ou não a validação de gramática no elemento.
style Define css inline.
tabindex Determina o estilo de tabulação do elemento.
title Pequeno texto exibido ao deixar o mouse sobre o elemento.
translate Define se um elemento pode ou não ser traduzido.

HTML5 API's

Existem diversas novas API's. Abaixo seguem as mais significativas segundo a W3C:

Geolocalização Utilizada para obter o posicionamento do usuário
Drag & Drop Permite arrastar e soltar, em tela, elementos programados para isso
Local Storage API que permite armazenamento de dados no cliente sem o uso de cookies
App Cache Permite a navegação offline de sua aplicação através de um arquivo de manifesto
Web Workers Execução de um javascript em background, que não afeta a performance da página
Server-Sent Events (SSE) Permite a uma pagina web atualizar dados com o servidor, automaticamente

Local Storage

Navegador... guarda meus dados?!

  • Armazena os dados no navegador
  • Criado para nao usar os Cookies para armazenamento de dados
  • Separado por origem (dominio e protocolo)
  • Permite iteração entre páginas
  • Tamanho muito maior que os cookies

Ah, então Local Storage é Cookie?!

NO! Local Storage != Cookie
